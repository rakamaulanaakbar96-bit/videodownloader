# Company Profile Website

A modern, responsive company profile website built with Next.js, Tailwind CSS, and Framer Motion.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open Browser**:
    Visit [http://localhost:3000](http://localhost:3000)

## Project Structure

-   `app/`: Source code for pages and components.
-   `app/components/`: Reusable UI components (Navbar, Hero, etc.).
-   `public/`: Static assets (images, icons).

## Customization

-   **Colors**: Edit `tailwind.config.ts` to change the `primary`, `secondary`, and `accent` colors.
-   **Content**: Edit the individual component files in `app/components/` to update text and images.
